# Troubleshooting Common Gmail Issues

A practical user guide that helps users resolve common Gmail problems quickly and effectively.

## What this guide covers
- Gmail sign-in and password recovery
- Two-step verification issues
- Lost or stolen security keys
- Sending and receiving email problems
- Attachment upload and download issues
- Blocking unwanted senders
- Frequently asked questions (FAQ)

## Audience
- Everyday Gmail users
- Non-technical users
- Support agents and helpdesk staff
- Technical writing portfolio reviewers

- ShareX / Snipping Tool (screenshots & annotations)
- GitHub (documentation hosting)
